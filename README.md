#### ZHLPHP 
> 自开发私有PHP MVC 框架  
> 小、精、简、快、易、多、强、广  

#### 运行条件 
> PHP+MYSQL/ORACLE  
> * php5.3以上版本
> * mysql5以上版本
> * php运行环境（如apache）
